# Heat Stress Prediction Tool

This repository hosts a Streamlit app for predicting afternoon spatial patterns of:

- Air temperature
- Dew point
- Solar radiation
- 2 m wind speed
- Surface pressure
- Wet Bulb Globe Temperature (WBGT)
- Black globe temperature
- Natural wet bulb temperature

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
